kruemelmann.github.io
=====================

This is my little blog about weird and hacky stuff;)

You can find it
---------------

[kruemelmann.github.io](https://kruemelmann.github.io)

Building
--------

You can build the page by running:
```bash
bazel build  //:site
```

Build wasm:
```bash
bazel build //:wasm
```


Usage
-----

To start the website locally in your browser:
```bash
bazel run //:site
```
